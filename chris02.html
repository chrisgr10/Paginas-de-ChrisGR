Ataques de Ciberseguridad

Ataque Descripción Impacto Imagen
Phishing Envío de correos electrónicos fraudulentos que parecen provenir de fuentes legítimas para robar datos personales. Robo de credenciales, pérdida financiera, compromiso de datos.
Malware Software malicioso diseñado para infiltrarse y dañar sistemas informáticos. Pérdida de datos, interrupción de servicios, espionaje.
Ataque DDoS Sobrecarga de servidores con tráfico masivo para interrumpir servicios. Caída de sitios web, pérdida de disponibilidad, costos operativos.
Ransomware Malware que cifra datos del usuario y exige un rescate para desbloquearlos. Pérdida de acceso a datos, costos de rescate, interrupción operativa.
Man-in-the-Middle (MitM) Intercepción de comunicaciones entre dos partes para espiar o manipular datos. Robo de información sensible, manipulación de datos, pérdida de confianza.
SQL Injection Inserción de código malicioso en consultas SQL para manipular bases de datos. Acceso no autorizado a datos, corrupción de bases de datos, filtración de información.
Credential Stuffing Uso de credenciales robadas para acceder a múltiples cuentas mediante ataques automatizados. Acceso no autorizado, robo de identidad, compromiso de cuentas.
